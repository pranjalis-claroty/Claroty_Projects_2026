# Claroty Projects 2026

![Language](https://img.shields.io/badge/Language-Python_100%25-blue.svg)


Welcome to the **Claroty Projects 2026** repository. This repository contains a collection of Python-based tools, scripts, and wizards designed to interact with, manage, and automate tasks within Claroty environments, specifically focusing on Continuous Threat Detection (CTD) and Programmable Logic Controllers (PLCs).

---

## 📁 Repository Structure

*   **`CTD_Wizard/`**
    Contains scripts and tools for automating deployments, configurations, or data extraction related to Claroty Continuous Threat Detection (CTD). 
    *See the internal README within this directory for specific usage instructions.*
*   **`PLC_Scripts/`**
    A collection of Python scripts dedicated to OT asset management, custom querying, or interacting safely with Programmable Logic Controllers (PLCs) in industrial environments.


---

*   **Python 3.x**: Ensure you have Python installed. You can download it from [python.org](https://www.python.org/).
*   **Virtual Environment (Recommended)**: It is highly recommended to run these scripts within an isolated virtual environment.
